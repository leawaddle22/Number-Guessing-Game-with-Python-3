# Number-Guessing-Game-with-Python-3
A Python CLI number guessing game with difficulty levels and replay system (input validation still in progress).

🎯 Number Guessing Game (Python)
📌 Description

This is a command-line Number Guessing Game built using Python.
The program randomly generates a number based on the selected difficulty, and the user attempts to guess it within a limited number of tries.

🚀 Features:
🎲 Random number generation

🎚️ Difficulty selection:
    Easy (1–20)
    Hard (1–100)
    
🔁 Maximum of 5 attempts per game

📉 Feedback system:
    "Too High"
    "Too Low"
    
🏆 Win/Lose result

🔄 Replay option

🛠️ Technologies Used
    Python 3
    Built-in random module
    
▶️ How to Run
    Make sure Python is installed
    Clone or download this repository
    Open terminal or command prompt
    Run the program:
    python main.py
    
🎮 How to Play
    Choose a difficulty level (easy or hard)
    The computer will generate a random number
    Enter your guess
    The program will tell you if your guess is too high or too low
    You have 5 attempts to guess correctly
    After the game ends, you can choose to play again
